---
description: A User Card UI component.
---

Displays a card of a user details. Uses the `User` component and breaks them down to cards.

## Basic usage

Use the user card component:

```tsx
<UserCard user={user} />
```

### Customize styles

Pass a `className` property to customize the card styling:

```tsx
import styles from './my-styles.module.scss';

<UserCard className={styles.userCard} user={user} />
```
