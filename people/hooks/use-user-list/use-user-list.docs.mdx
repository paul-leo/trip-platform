A react hook for retriving user data from the Pied Piper platform API.

## Get started

Install and use the hook in your components:

```tsx
import { useUserList } from '@nore-work/people.hooks.use-user-list';

export function UserCount() {
  const users = useUserList();
  if (!users) return null;

  return <div>found {users.length} users</div>;
}
```

## Mock data

The component returns mock data for testing and preview purposes when running in a mock mode:

```tsx
import { useUserList } from '@nore-work/people.hooks.use-user-list';

<MockProvider>
  <UserCount />
</MockProvider>
```
